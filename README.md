# Gympass Style API

## English Version

### Overview
The Gympass Style API is designed to provide a seamless integration for managing gym memberships, classes, and user activities. This API allows developers to create, read, update, and delete information related to gym services.

### Features


### Installation
To install the Gympass Style API, follow these steps:
1. Clone the repository:
  ```bash
  git clone https://github.com/raulneto90/gympass-style-api.git
  ```
2. Navigate to the project directory:
  ```bash
  cd gympass-style-api
  ```
3. Install dependencies:
  ```bash
  npm install
  ```

### Usage
To start the server, run:
```bash
npm start
```
The API will be available at `http://localhost:3333`.
